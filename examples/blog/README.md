# Blog

A example web application that created by stages. \
See Roadmap.md for all the stages of this project.

## Requirements

- yarn
- node

## Get Started

For get started run the following commands in the terminal:

```bash
yarn install
yarn run start
```

Open in browser the site <http:\\localhost:3000>
